# おうちで参拝 LINE bot

## １、概要
直接神社を訪れなくても、その神社のご利益を受けることができるアプリ

### ターゲットユーザー
神社のご利益を受けたいが、物理的・地理的制限から神社を訪れることができないひと。

### アプリ内で参拝可能な神社
- 今は受験シーズン
- 合格祈願神社　人気ランキングTOP６
1. 太宰府天満宮（福岡県）
2. 北野天満宮（京都府）
3. 湯島天神（東京都）
4. 防府天満宮（山口県）
5. 出雲大社（島根県）
6. 善光寺（長野県）

### アプリの概要
- 機能を４つ実装
    1. お参り
    2. お守り購入
    3. おみくじ購入
    4. 履歴取得
    （機能の詳細はアプリのデモで説明します。）

## ２、使用した機能

### 銀行API

1. 銀行の口座金額を確認する　**残高照会**　
2. お参り、お守り、おみくじ　**振込依頼**　
3. 振り込んだ後に使用する　**振込依頼結果照会**　
4. 振り込んだ履歴　**入出金明細照会**　

### LINEbot

1. TOP画面にメニューを大きく表示　**リッチメニュー**　
2. メニュー選択後分岐に使用　**カードタイプメッセージ・クイックリプライ**　の２種類　
3. 選んだあとに自動で返信　**自動応答メッセージ**
